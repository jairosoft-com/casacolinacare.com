## Codebase Patterns
- Fonts: Inter (body/`font-sans`) and Playfair Display (headings/`font-heading`) configured via `next/font/google` CSS variables in layout.tsx and Tailwind fontFamily config
- Headings h1-h3 automatically get `font-heading` via globals.css base layer
- The `font-heading` class is the only change needed for logo font — no new font packages, CSS, or Tailwind config required
- Footer is a server component — no `'use client'` needed
- E2E tests targeting the site footer should use `page.getByRole('contentinfo')` not `page.locator('footer')` — there are multiple `<footer>` elements on the page
- ADO closing tasks: do NOT include RemainingWork=0 — ADO rule rejects empty Remaining Work on Closed state. Only set CompletedWork.
