# Glossary

Workplace shorthand, acronyms, and internal language for the Casa Colina Care project.

## Acronyms

| Term | Meaning | Context |
|------|---------|---------|
| RSC | React Server Component | Next.js default rendering model |
| SSG | Static Site Generation | How all 4 pages render |
| SSR | Server-Side Rendering | Not used currently |
| CTA | Call to Action | Banner component on multiple pages |
| CVA | Class Variance Authority | Button variant styling |
| OG | Open Graph | Social sharing metadata |
| JSON-LD | Linked Data JSON | Structured data for SEO |
| HSL | Hue Saturation Lightness | Color system in CSS variables |
| PRD | Product Requirements Document | In prd/ directory |
| BRD | Business Requirements Document | Template in skills |
| TDS | Technical Design Specification | Jodex skill output |
| US | User Story | e.g. US-001, US-012 |
| E2E | End-to-End | Playwright tests |
| CVE | Common Vulnerabilities and Exposures | Security fixes |
| PR | Pull Request | GitHub workflow |
| ADO | Azure DevOps | Project management & work tracking |
